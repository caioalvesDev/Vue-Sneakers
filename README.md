# Vue Sneakers 👟

# Hospedado na [Vercel](https://vercel.com/)
[Visualizar Vue-Sneakers](https://vue-sneakers-chi.vercel.app/)

## Como começar?

1. Clone o repositório para o seu PC
2. `npm install`
3. `npm run dev`
4. Pronto!


**⚙️ Tecnologia (stacks):**
- Vue 3
- Vue Router
- TailwindCSS
- Axios
- @formkit/auto-animate
- Composition API / Options API
- ESLint
- Prettier
